第一篇还是给如何hexo在github发布你的BLOG

安装各种环境

因为各系统不同就不做赘述，如果你是用Mac，必须要安装Node,Xcode自带git。
当然你还需要Github账号，并且建立你的BLOG仓库。注意BLOG仓库必须用{username}.github.io命名，否则发布后无法访问。

特别讲一下SSH Github

如果你需要SSH Github，你首先要创建一个SSH Key.
可以在终端先查询下你的SSH Key的情况

把你的SSH key 添加到ssh-agent, 编辑config文件

然后添加你的私钥和储存密码（如果有）

然后让我们做一个测试

输出

就表示成功的连上github了

安装HEXO

对除了环境外，就是全局安装HEXO

安装完成后就是新建一个文件目录并初始化一下

要把hexo发布到github，那么需要配置下 _config.yml的deploy

注意以上冒号后面均有空格！否则发布一定不成功。
那hexo常用命令如下：
hexo clean 清除发布目录
hexo generate / hexo g 创建发布内容
hexo deploy 发布
hexo server 本地服务
hexo new 创建文章
hexo new page 创建页面

具体不赘述，可以查看hexo官方文档
